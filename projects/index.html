Analyticator

I built Analyticator because it was almost painful to add Google Analytics event tracking to an application that I was developing because testing meant looking at the requests made when I clicked on something and seeing if the details looked right.

While it does currently track events that are sent accurately, it is under active development. As I find something in Google Analytics that I'd like to track in development, I will add it to Analyticator.

For more information and directions for use, check out Analyticator on Github.

Uploader

Uploader makes sending files securely very simple. The application allows you to upload a file and then encrypts it based on the password that you enter when uploading files and stores it securely in Amazon's S3.

When you retrieve the files, the server attempts to decrypt it with your supplied password (on the download page) and then returns it to you!

PermitMe

PermitMe is an application designed to make the permitting process much easier for new businesses. PermitMe was started at Startup Weekend Houston!

ControlVC

ControlVC is an application that allows users to manage many videoconferencing devices through a simple web interface. ControlVC has since been open-sourced at Github so feel free to check it out and leave any feedback you have!

Cities Index

The cities index allowed my wife and I to compare cities more effectively to determine what we were looking for in a city and what cities fit the criteria we wanted.

Because I was trying to compare so many cities at one time, I decided to use data from multiple sources to describe various cities but the primary source was the US Census API because of the data available in describing cities.
